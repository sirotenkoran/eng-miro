Eng Lessons — Miro App

Static host for the Miro side-panel app that wraps interactive Speakout A2 English lessons.

Endpoints

  • /app.js — SDK URI (registered in Miro Developer Dashboard)
  • /panel.html — side panel UI (opened by Miro)
  • /lead-in.html — the lesson itself (standalone-capable)
  • /lessons.json — lesson catalogue
